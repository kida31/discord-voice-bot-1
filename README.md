# Features

### Automatic Member Announcements
The bot automatically generates custom notifications whenever a member joins or leaves the server.  
This ensures moderators and community members always stay informed without needing to monitor activity manually.

### Text-to-Speech Function (TTS)
The bot can speak any user-provided text in a voice channel.
